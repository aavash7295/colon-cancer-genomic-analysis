# Colon Cancer Genomic Analysis (BINF702)

## Overview
This project analyzes colon cancer gene expression data from the NCI60 dataset to identify genomic instability patterns and potential biomarker genes.

## Methods
- Z-score normalization
- Market marker gene selection
- Outlier detection (|Z| > 2)
- PCA & t-SNE
- K-means & hierarchical clustering
- Self-Organizing Maps (SOM)
- Generalized Additive Models (GAM)
- Random Forest & Neural Network validation

## Key Results
- Identified 5 high-consistency driver genes
- Detected 677 background outlier genes
- Found 3 distinct patient clusters
- Cross-model validation confirmed key biomarkers
